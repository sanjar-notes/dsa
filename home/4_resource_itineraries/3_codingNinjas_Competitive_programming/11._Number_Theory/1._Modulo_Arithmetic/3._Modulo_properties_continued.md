What about negative numbers, how does modulo behave?

* For positive  - a%p ∈ [0, p-1]
* If a%p comes out of negative, we take (a%p + p)%p. Works for all cases.


*****

**Exponentiation**
If a ≡ b(modN), then a^k^ ≡ b^k^(modN) any positive integer k.
